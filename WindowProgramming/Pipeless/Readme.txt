구현한 기능
랜덤 블록 떨어지기
다음에 떨어질 블록 띄워주기
블록 비트맵으로 띄우기
키보드 입력으로 좌우 움직이기
아래 화살표키 누르면 빠르게 떨어짐
r키를 누르면 블록 회전
s키를 누르면 일시 정지/일시정지 해제
q키를 누르면 종료
z 대 x 중 c 소 순으로 블록 크기 조절 및 초기화
블록 하강속도가 일정 속도까지 일정 시간마다 빨라짐
화면 천장까지 블록이 쌓일경우 게임 오버 메시지 출력 및 게임 정지
보드 재설정으로 다시 시작할 수 있음
아래 블록 삭제시 위의 블록들이 차례로 아래로 떨어짐

구현하지 못한 기능
블록 삭제
ㅡ자형 블록이 가로 한줄로 쌓였을 경우에 삭제 기능만 구현

Bitmap 파일을 같이 넣어두었으나 실행 시에 주소가 달라서 실행이 안될겁니다.
주소를 파일이 들어있는 곳으로 바꿔주면 실행이 됩니다.
